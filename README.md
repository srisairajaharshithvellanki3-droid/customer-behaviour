# Customer Shopping Behavior Analysis

## Overview

This project presents an end-to-end data analytics workflow focused on analyzing customer shopping behavior. The objective is to uncover purchasing patterns, customer preferences, and key business insights using Python, SQL, and Power BI.

The project demonstrates skills in data cleaning, exploratory data analysis (EDA), database querying, dashboard development, and business reporting.

---

## Dataset

* **Name:** Customer Shopping Behavior Dataset

* **Size:** ~3,900 rows × 18 columns

* **Description:**
  The dataset contains customer-level transaction and behavioral data, including demographics, purchasing habits, and product preferences.

* **Key Features:**

  * **Customer Info:** Age, Gender, Location
  * **Purchase Details:** Item Purchased, Category, Purchase Amount (USD)
  * **Behavioral Data:** Frequency of Purchases, Previous Purchases
  * **Preferences:** Payment Method, Shipping Type, Subscription Status
  * **Engagement Metrics:** Review Rating, Promo Code Usage, Discounts

## Project Steps

### 1. Data Loading

* Imported dataset using Pandas
* Checked data structure, data types, and missing values

### 2. Exploratory Data Analysis (EDA)

* Analyzed customer demographics (age, gender distribution)
* Examined purchasing trends across categories and seasons
* Identified high-spending customer segments
* Visualized relationships between variables (e.g., purchase frequency vs spending)

### 3. Data Cleaning

* Handled missing or inconsistent values
* Removed duplicates (if any)
* Standardized categorical values (e.g., payment methods, locations)
* Converted data types where necessary

### 4. SQL Analysis

* Loaded cleaned data into a relational database
* Performed queries to:

  * Calculate total and average purchase amounts
  * Identify top-performing product categories
  * Analyze customer segments by spending behavior
  * Evaluate impact of discounts and promo codes

### 5. Dashboard Development

* Built an interactive Power BI dashboard including:

  * KPIs (Total Revenue, Average Spend, Customer Count)
  * Sales by Category and Season
  * Customer segmentation (age, gender, location)
  * Purchase frequency and behavior insights
  * Filters for dynamic analysis

## Results & Insights

* Clothing and footwear categories contribute significantly to total sales
* Customers with higher previous purchases tend to spend more
* Discounts and promo codes positively influence purchase behavior
* Certain age groups show higher purchase frequency and engagement
* Seasonal trends impact product demand
